# m3uEdit
convert iptv m3u to csv and back

### Usage

```python3 /path/to/m3u_to_csv.py '/path/to/input.m3u' '/path/to/output.csv'```

```python3 /path/to/csv_to_m3u.py '/path/to/input.csv' '/path/to/output.m3u'```
